Create OI for curvature drive (drives like a car). One throttle controls forward-backward speed, like arcade, but the other controls curvature instead of rotation. Ramping is still applied to rotation instead of curvature.

Parameters

drive

The drivetrain

xThrottle

Throttle to get forward-backward movement

rotThrottle

Throttle to get rotation

xRamp

Used for limiting forward-backward acceleration

rotRamp

Used for limiting rotational acceleration

turnInPlace

When this returns true, turns in place instead of turning like a car
