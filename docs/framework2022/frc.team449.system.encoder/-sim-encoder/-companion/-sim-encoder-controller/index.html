SimEncoderController

public final class SimEncoderController
Content copied to clipboard

Used to control {@link SimEncoder}s. Only one {@link SimEncoderController} should be used per {@link SimEncoder}.

Make sure to only create SimEncoders during simulation. If you create, say, a {@link NEOEncoder}, during simulation and try to use a {@link SimEncoderController} to control it, the {@link SimEncoderController} constructor will throw an error.
