BackupEncoder

A wrapper to use when you have one external encoder that's more accurate but may be unplugged and an integrated encoder that's less accurate but is less likely to be unplugged.

If the primary encoder's velocity is 0 but the integrated encoder's is above a given threshold, it concludes that the primary encoder is broken and switches to using the fallback/integrated encoder.
