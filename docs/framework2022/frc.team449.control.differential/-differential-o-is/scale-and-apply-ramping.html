fun scaleAndApplyRamping(    wheelSpeeds: DifferentialDrive.WheelSpeeds,     kinematics: DifferentialDriveKinematics,     maxSpeed: Double,     xRamp: SlewRateLimiter,     rotRamp: SlewRateLimiter): ChassisSpeeds

Scales differential drive speeds from -1, 1 using {@code maxSpeed}, then applies ramping.

Do note that although this is given a {@link DifferentialDriveWheelSpeeds} object, the ramping isn't applied to the left and right side but to the linear and rotational velocity using a {@link ChassisSpeeds} object.

Parameters

wheelSpeeds

The left and right wheel speeds

kinematics

Kinematics object used for turning differential drive wheel speeds to chassis speeds

maxSpeed

Max linear speed for the drivetrain

xRamp

Used for limiting linear/forward-back acceleration

rotRamp

Used for limiting rotational acceleration
